RlassoModels implements rigorous Lasso for estimation and inference in high-dimensional datasets. It aims to provide a Python alternative to the existing stata packages lassopack and pdslasso (Ahrens, Hansen & Schaffer, 2018, 2020) and the R package hdm (Chernozkukov, Hansen & Spindler, 2016).

from sklearn.datasets import make_regression
from rlassomodels import Rlasso

# Generate data.
X, y = make_regression(n_samples=100, n_features=80, n_informative=5)

# Define the model and fit to data
model = Rlasso()
model.fit(X, y)
